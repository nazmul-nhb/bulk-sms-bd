`bulk-sms-bd` is a lightweight, clean, and type-safe JavaScript/TypeScript library for integrating with the [bulksmsbd.net](https://bulksmsbd.net/) SMS gateway. It supports sending single, bulk, and personalized SMS messages using a promise-based API with comprehensive error handling and complete TypeScript definitions.

---

## Features

- **Comprehensive SDK Support**: Zero-dependency library, fully bundled with helper utilities.
- **Multiple API Channels**:
  - **One-to-One**: Send a single SMS to a specific number.
  - **One-to-Many**: Send the same SMS to multiple numbers in a single API call.
  - **Many-to-Many**: Send different custom SMS messages to different numbers simultaneously.
- **Robust Error Handling**: Distinct `BulkSmsError` instances mapping all API error codes (`1001`-`1032`) from the bulksmsbd.net gateway.
- **Flexible Package Formats**: Includes ESM and CommonJS formats out-of-the-box.
- **Strictly Typed**: Full TypeScript support with internal type declarations.

## Quick Examples

### Initialization

```typescript
import { BulkSmsClient } from 'bulk-sms-bd';

const smsClient = new BulkSmsClient({
  bulkSmsApiKey: 'YOUR_API_KEY',     // API key provided by bulksmsbd.net
  bulkSmsSenderId: 'YOUR_SENDER_ID'  // Your registered Sender ID / Mask
  
  // Optional base url if needed, default to 'https://bulksmsbd.net/api'
  // bulkSmsBaseApi: 'https://custom.bulksmsbd.net/api'
});
```

### 1. Single SMS (One-to-One)

Send a single message to a phone number.

```typescript
try {
  const res = await smsClient.sendSMS('88016XXXXXXXX', 'Hello, this is a test message!');
  console.log('Success:', res.success_message); // "SMS Sent Successfully"
} catch (error) {
  if (error instanceof BulkSmsError) {
    console.error(`Error Code ${error.code}: ${error.message}`);
  }
}
```

*Alternative usage using `oneToOne` alias:*

```typescript
const res = await smsClient.oneToOne('88016XXXXXXXX', 'Hello, this is a test message!');
```

### 2. Same SMS to Multiple Numbers (One-to-Many)

Send the exact same message to multiple phone numbers at once.

```typescript
try {
  const recipients = ['88016XXXXXXXX', '88017XXXXXXXX', '88018XXXXXXXX'];
  const res = await smsClient.sendSMS(recipients, 'Hello everyone, this is a broadcast message!');
  console.log('Success:', res.success_message);
} catch (error) {
  console.error('Failed to send bulk SMS:', error);
}
```

*Alternative usage using `oneToMany` alias:*

```typescript
const res = await smsClient.oneToMany(recipients, 'Hello everyone!');
```

### 3. Different SMS to Different Numbers (Many-to-Many)

Send personalized messages to different recipients.

```typescript
import type { Message } from 'bulk-sms-bd/types';

try {
  const messages: Message[] = [
    { to: '88016XXXXXXXX', message: 'Hello Alice, your otp is 1234.' },
    { to: '88017XXXXXXXX', message: 'Hello Bob, your otp is 5678.' }
  ];
  
  const res = await smsClient.sendSMS(messages);
  console.log('Success:', res.success_message);
} catch (error) {
  console.error('Failed to send personalized SMS:', error);
}
```

*Alternative usage using `manyToMany` alias:*

```typescript
const res = await smsClient.manyToMany(messages);
```

## API Reference

### `BulkSmsClient`

The primary class used to interface with the bulksmsbd.net API.

#### Constructor

```typescript
constructor(configs: BulkSmsBdConfig)
```

Creates a new `BulkSmsClient` instance.

**Parameters:**

- `configs`: `BulkSmsBdConfig` (Required)
  - `bulkSmsApiKey`: `string` (Required) - API Key from the bulksmsbd.net dashboard.
  - `bulkSmsSenderId`: `string` (Required) - Registered Sender ID / Mask.
  - `bulkSmsBaseApi`: `string` (Optional) - Override base API URL. Defaults to `'https://bulksmsbd.net/api'`.

*Throws a `BulkSmsError` with code `1003` if configs are empty or missing `bulkSmsApiKey` / `bulkSmsSenderId`.*

#### `sendSMS()`

Flexible method to dispatch SMS based on arguments provided.

```typescript
async sendSMS(to: string, message: string): Promise<SuccessResponse>;
async sendSMS(to: string[], message: string): Promise<SuccessResponse>;
async sendSMS(messages: Message[]): Promise<SuccessResponse>;
```

#### `oneToOne()`

Wrapper for `sendSMS()` for single messages.

```typescript
async oneToOne(to: string, message: string): Promise<SuccessResponse>
```

#### `oneToMany()`

Wrapper for `sendSMS()` for single messages to multiple recipients.

```typescript
async oneToMany(to: string[], message: string): Promise<SuccessResponse>
```

#### `manyToMany()`

Wrapper for `sendSMS()` for sending multiple distinct messages.

```typescript
async manyToMany(messages: Message[]): Promise<SuccessResponse>
```

---

### `BulkSmsError`

Extends the native `Error` class to wrap any failures thrown by the SDK or returned by the gateway.

```typescript
class BulkSmsError extends Error {
  code: ErrorCode;
  constructor(message: string, code: ErrorCode);
}
```

**Properties:**

- `message`: `string` - Error message details.
- `code`: `ErrorCode` (number) - Code mapping to the gateway issue (e.g. `1001` - `1032`) or SDK errors (e.g. `1003` - validation, `1005` - fetch failure).

---

### `isMessageObject()`

Type guard helper to validate if a value meets the `Message` structure.

```typescript
function isMessageObject(value: unknown): value is Message
```

## Constants Reference

### API Endpoints

- Base API URL: `https://bulksmsbd.net/api`
- Single / Bulk Endpoint: `/smsapi` (uses HTTP POST)
- Personalized Endpoint: `/smsapimany` (uses HTTP POST)

### API Status & Error Codes Map

| Code   | Status/Error Description                                                |
| ------ | ----------------------------------------------------------------------- |
| `202`  | SMS Sent Successfully                                                   |
| `1001` | Invalid Phone Number                                                    |
| `1002` | Sender ID Not Correct/Sender ID is Disabled                             |
| `1003` | All Fields Required, Please Contact Administrator                       |
| `1005` | Internal Error                                                          |
| `1006` | Balance Validity Not Available                                          |
| `1007` | Balance Insufficient                                                    |
| `1011` | User ID Not Found                                                       |
| `1012` | Masking SMS Must Be Sent in Bengali                                     |
| `1013` | Sender ID Not Found Gateway by API Key                                  |
| `1014` | Sender Type Name Not Found Using this Sender by API Key                 |
| `1015` | Sender ID Not Found Any Valid Gateway by API Key                        |
| `1016` | Sender Type Name Active Price Info Not Found By this Sender ID          |
| `1017` | Sender Type Name Price Info Not Found By this Sender ID                 |
| `1018` | The Owner of this (username) Account is Disabled                        |
| `1019` | The (Sender Type Name), Price of this (username) Account is Disabled    |
| `1020` | The Parent of this Account is Not Found                                 |
| `1021` | The Parent Active (Sender Type Name) Price of this Account is Not Found |
| `1031` | Your Account Not Verified, Please Contact Administrator                 |
| `1032` | IP Not Whitelisted                                                      |

> [!NOTE]
> The original list can be found here: <https://bulksmsbd.net/developers>

## Type Definitions

```typescript
interface Message {
  to: string;
  message: string;
}

interface BulkSmsBdConfig {
  bulkSmsBaseApi?: string;
  bulkSmsApiKey: string;
  bulkSmsSenderId: string;
}

interface SuccessResponse {
  response_code: 202;
  success_message: string;
}

interface ErrorResponse {
  response_code: ErrorCode;
  error_message: string;
}

type BulkSmsResponse = SuccessResponse | ErrorResponse;

type SuccessCode = 202;

type ErrorCode = 1001 | 1002 | ... | 1032;
```
